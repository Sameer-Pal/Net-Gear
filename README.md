## Setup Steps

**Clone the repository**:

```bin
git  clone https://github.com/yourusername/lancer-unit.git
cd lancer-unit
```
**Install dependencies**:
```bin 
npm install
```
**Start the development server**:
```bin 
npm start
```
Open your browser and visit:
```bin
http://localhost:3000 
```
